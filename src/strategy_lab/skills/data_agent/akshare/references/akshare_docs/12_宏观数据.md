## 中国宏观

#### `macro_cnbs`

**描述**: 中国国家金融与发展实验室-中国宏观杠杆率数据

**限量说明**: 单次返回所有历史数据


**输入参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| - | - | - |


**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| 年份 | object | 日期, 年-月 |
| 居民部门 | float64 | - |
| 非金融企业部门 | float64 | - |
| 政府部门 | float64 | - |
| 中央政府 | float64 | - |
| 地方政府 | float64 | - |
| 实体经济部门 | float64 | - |
| 金融部门资产方 | float64 | - |
| 金融部门负债方 | float64 | - |

**示例代码**:

```python

```
