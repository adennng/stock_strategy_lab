## 高频数据

#### `hf_sp_500`

**描述**: 获取标普 500 指数的分钟数据, 由于数据量比较大, 需要等待, 由于服务器在国外, 建议使用代理访问

**数据源**: https://github.com/FutureSharks/financial-data


**输入参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| year | str | year="2017"; 只能获取 2012-2018 年的数据 |


**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| date | object | 日期时间 |
| open | float64 | 开盘价 |
| high | float64 | 最高价 |
| low | float64 | 最低价 |
| close | float64 | 收盘价 |

**示例代码**:

```python
import akshare as ak

hf_sp_500_df = ak.hf_sp_500(year="2017")
print(hf_sp_500_df)
```
