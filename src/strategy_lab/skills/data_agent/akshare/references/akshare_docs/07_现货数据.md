# AKShare 现货数据

> 数据来源: [AKShare 文档](https://akshare.akfamily.xyz)

本文档包含 **AKShare 现货数据** 分类下的所有 AKShare 接口函数介绍。

---


## 现货数据

#### `spot_price_qh`

**描述**: 99 期货-数据-期现-现货走势

**数据源**: https://www.99qh.com/data/spotTrend

**限量说明**: 单次返回指定 symbol 的所有历史数据；由于数据源限制，只能获取个别品种


**输入参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| symbol | str | symbol="螺纹钢"; 可以通过 ak.spot_price_table_qh() 获取品种表 |


**输出参数**

| 名称 | 类型 | 描述 |
|------|------|------|
| 日期 | object | - |
| 期货收盘价 | float64 | - |
| 现货价格 | float64 | - |

**示例代码**:

```python
import akshare as ak

spot_price_qh_df = ak.spot_price_qh(symbol='螺纹钢')
print(spot_price_qh_df)
```
