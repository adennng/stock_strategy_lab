# signal_direct_baseline

## 策略目标

第一轮先验证“直接按信号层仓位执行”的组合效果。  
本策略把每个资产的 `signal_targets` 视为单资产目标仓位，然后只施加组合层的基础约束：最大总仓位、单资产上限、换手限制和平滑。

## 画像依据

- 信号层每个资产已经完成单资产策略训练。
- 信号层输出的 `S_i` 可以理解为该资产单独视角下愿意参与的目标仓位。
- 如果多个资产信号同时较强，组合层需要用 `max_gross` 和 `max_weight` 控制总风险。
- 该版本用于建立信号层直用基线，后续再与预算层直用、预算主导修正、双证据融合等版本比较。

## 核心结构

```text
W_raw_i = S_i
W_cap_i = min(W_raw_i, max_weight)
若 sum(W_cap) > max_gross，则按比例缩放
应用 rebalance_speed 和 max_turnover_per_day
```

## 关键参数

- `max_gross = 1.00`：允许信号层在机会足够时接近满仓，但不加杠杆。
- `max_weight = 0.25`：控制单资产集中度。
- `rebalance_speed = 0.80`：较快跟随信号层变化。
- `max_turnover_per_day = 0.50`：限制日换手。
- `use_budget_cap = false`：第一版不使用预算层上限，便于观察信号层本身效果。

## 预期优势

- 最直接地检验信号层整体质量。
- 策略逻辑简单，便于作为后续版本对照组。
- 不会因为预算层和信号层相乘导致仓位被过度压低。

## 主要风险

- 如果信号层整体过于保守，组合可能长期低仓。
- 如果多个信号策略同时误判，缺少预算层保护。
- 如果信号层换手较高，组合换手可能偏高。

## 后续可改方向

- 加入预算层作为上限或偏好。
- 对高可靠信号资产允许更高仓位。
- 对低可靠或风险门控型信号做更强折扣。
- 引入闲置资金再分配。
- 引入分市场状态下的总仓位控制。
